# Task 3.19 Defense PPT Outline

以下内容可直接转成答辩 PPT。建议 10 到 12 页。

## 1. 标题页

题目：3.19 轮式人形机器人垃圾分类投放

项目：MDA IsaacLab Task319 Garbage Sorting

关键词：IsaacLab、Kuavo 轮式双臂、RGB-D、Qwen/VLM、YOLO11、Nav2、抓取投放

## 2. 任务目标

- 面向智慧城市垃圾分类高频任务。
- 机器人识别桌面垃圾，判断四分类类别。
- 规划抓取姿态，移动到对应垃圾桶并投放。
- 输出分类准确率、抓取成功率、投放准确率。
- 设计干扰样本评估鲁棒性。

## 3. 仿真场景

- 仿真平台：IsaacLab/Isaac Sim。
- 机器人：Kuavo 轮式双臂模型，头部 RGB-D，相机分辨率主线 1280x960。
- 场景：桌面、4 个带颜色标识垃圾桶、10 个垃圾物体。
- 垃圾桶：蓝色可回收、绿色厨余、红色有害、灰色其他。

## 4. 场景物体和类别

| 类别 | 物体 |
| --- | --- |
| 可回收物 | cracker box、sugar box、tomato soup can、mustard bottle |
| 厨余垃圾 | banana、potted meat can 食物残留干扰样本 |
| 有害垃圾 | bleach cleanser、battery block |
| 其他垃圾 | foam brick、mug |

说明：`potted meat can` 是可回收/厨余语义冲突样本，用于测试污染可回收物的鲁棒性。

## 5. 系统架构

```text
头部彩色图像与深度图
-> VISS v28 与 Qwen 优先识别
-> YOLO11 局部区域分割
-> Qwen/视觉语言模型四分类
-> 三维定位和顶部抓取
-> 动态桌边站位
-> Nav2 NavigateToPose
-> 右臂抓取或辅助抓取
-> 分类桶 Nav2
-> 右夹爪桶口释放
```

## 6. 视觉识别算法

- Qwen 全局识别候选，避免只靠固定类别检测。
- YOLO11 在 Qwen 候选区域内做精分割，输出目标区域。
- Qwen/视觉语言模型对局部图像做开放词汇四分类。
- 当前帧 RGB-D 根据目标区域反投影到世界坐标。
- 分类、位置、抓取点绑定同一目标身份，避免跨物体错配。

## 7. 抓取姿态生成

- 默认主线使用深度图顶部抓取。
- 抓取点：点云 XY 中心和物体顶部高度。
- 姿态：顶部垂直接近，夹爪开口根据 RGB-D 点云短轴估计。
- cuRobo 规划到物体上方预备抓取点，最后阶段做垂直下降和闭合。
- GraspNet Baseline 保留为诊断，不作为默认主线。

## 8. 移动和投放

- 机器人先在观察位看全桌。
- 根据目标 3D 位置生成动态桌边站位。
- Nav2 导航到站位后重新拍照，降低底盘残差影响。
- 抓取后导航到桶侧等待位置。
- 右夹爪末端移到桶口上方再释放，避免直接瞬移物体。
