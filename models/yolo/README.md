# YOLO Weights

Large YOLO `.pt` files are intentionally not tracked in Git.

Download public fallback weights with:

```bash
curl -L -o models/yolo/yolov8m-seg.pt \
  https://github.com/ultralytics/assets/releases/download/v8.3.0/yolov8m-seg.pt
```

See `docs/task319_delivery/README.md` for the full model setup.
