# VISS Model Weights

Large `.pt` weights are intentionally not tracked in Git.

For Task319 mainline perception, place the project-trained weight here:

```text
viss/models/yolo11s-seg-best.pt
```

Optional fallback weights can also be restored here from the model bundle:

```text
viss/models/best_seg.pt
viss/models/yolo11s-seg.pt
viss/models/yoloe-26s-seg.pt
viss/models/yoloe.pt
```

See `docs/task319_delivery/README.md` for download and placement steps.
